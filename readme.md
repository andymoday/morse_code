## Morse Code Translator

Takes input from user (string) and converts into Morse Code

Acceptable characters:

<ul>
    <li>A-Z</li> 
    <li>a-z</li>  
    <li>0-9</li> 
    <li>[ . , ? ' ! / ( ) & : ; = + - _ " $ @ ]</li> 
</ul>

### Source for data: 
https://morsecode.world/international/morse.html
